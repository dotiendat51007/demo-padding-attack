# demo-padding-attack
Để chạy được code cần cài thư viện Cryptography.
Thư viện Cryptography là một thư viện Python cung cấp các công cụ để mã hóa, giải mã và bảo vệ dữ liệu một cách an toàn, dựa trên các nguyên lý của Cryptography.
- Cách cài thư viện cryptography trong Python:
1. Cài bằng pip
 Mở terminal/cmd:
 pip install cryptography
 Hoặc nếu dùng Python 3:
 pip3 install cryptography
2. Kiểm tra cài thành công
 import cryptography
 print("OK")
